Apple Календарь (iCloud)

Синхронизируйте задачи на iPhone, iPad, Mac

1

Скопируйте iCal-ссылку

Используйте ту же ссылку, что и для Google Календаря

2

На Mac: Откройте Календарь

Файл → Новая подписка на календарь → вставьте ссылку

3

На iPhone/iPad: Настройки → Календарь → Аккаунты → Добавить → Подписной календарь

Настройки → Календарь → Аккаунты → Добавить → Подписной календарь → вставить ссылку

Outlook / Microsoft 365

Синхронизация с Outlook, Hotmail, Exchange

1

Скопируйте iCal-ссылку

Используйте ту же ссылку

2

Откройте Outlook в браузере

Перейдите в Календарь → Добавить календарь → Подписаться из Интернета

3

Вставьте ссылку и назовите календарь

Например: "CRM Задачи"
